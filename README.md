# Next.jsでConduitのフロントエンドを作成しました。

## 起動方法！
1. npm run dev でフロント側をlocalhost:3000で立ち上げます。

2. 前回まで作成した[Rails API]()は
``rails s -p 3001``で起動し、<br>3001番ポートで起動してください。
